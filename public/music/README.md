# 背景音乐说明

## 使用方法

将您的婚礼背景音乐文件命名为 `wedding-bgm.mp3` 并放置在此目录下。

## 推荐格式

- **格式**: MP3（推荐）或 WAV
- **比特率**: 128kbps - 320kbps
- **时长**: 3-5 分钟（会循环播放）
- **音量**: 建议预先调整音频音量，避免过大或过小

## 推荐曲目风格

婚礼邀请函适合的背景音乐：

1. **轻柔浪漫**: 钢琴曲、小提琴曲
2. **温馨甜蜜**: 吉他弹唱、轻音乐
3. **经典婚礼**: 婚礼进行曲、爱的礼赞

## 注意事项

- 浏览器通常会阻止音频自动播放
- 本应用通过终端页面的"点击进入"来触发音乐播放
- 音乐会在用户点击后以淡入效果开始播放
- 如果不需要背景音乐，可以删除此文件或在代码中禁用

## 在线音乐资源

如果您想使用在线音乐，可以在 `TerminalPage.vue` 中修改音乐 URL：

```typescript
playMusic('https://example.com/your-music.mp3', 0, true)
```

## 禁用背景音乐

如果不需要背景音乐，请在 `src/components/pages/TerminalPage.vue` 中注释或删除 `playBackgroundMusic()` 调用。
